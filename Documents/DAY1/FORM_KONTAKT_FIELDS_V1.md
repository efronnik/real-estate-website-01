# Form Kontakt Fields v1

## Punkt 3 - finalne pola formularza `kontakt` (zatwierdzone)

### Required
- `full_name` - imie i nazwisko
- `phone` - telefon
- `consent_data` - zgoda na przetwarzanie danych

### Optional
- `email` - e-mail
- `message` - krotki opis zapytania
- `city_or_district` - miasto / dzielnica
- `preferred_contact_time` - preferowana godzina kontaktu

### Technical hidden fields
- `lead_type` = `kontakt`
- `source_page` = nazwa strony
- `language` = `pl`
- `utm_source`
- `utm_medium`
- `utm_campaign`
- `created_at` (na backendzie)
